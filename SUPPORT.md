## Before Opening an Issue

1. Check the [FAQ](FAQ.md) for common questions
2. Search existing [issues](https://github.com/hassan-ali/gold-elite-2-0/issues) for duplicates
3. Make sure you're running the latest version
4. Check the VS Code Developer Tools console (Help → Toggle Developer Tools) for errors

## Reporting Bugs

Please use the [Bug Report template](https://github.com/hassan-ali/gold-elite-2-0/issues/new?template=bug_report.md) and include:

- VS Code version (Help → About)
- OS and version
- Extension version
- Steps to reproduce
- Expected behavior
- Actual behavior
- Developer Tools console output
