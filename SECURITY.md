# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 0.4.x | Yes |
| 0.3.x | No |
| < 0.3 | No |

## Reporting a Vulnerability

Gold Elite takes security seriously. Since this extension makes zero network requests and stores all data locally, the attack surface is minimal.

If you discover a security vulnerability, please report it privately by opening a [GitHub Security Advisory](https://github.com/hassan-ali/gold-elite-2-0/security/advisories/new).

Do not disclose the vulnerability publicly until it has been addressed.

## What to Expect

- You will receive an acknowledgment within 48 hours
- We will investigate and provide a timeline for a fix
- A fix will be released as a patch version
- You will be credited in the release notes (unless you prefer to remain anonymous)

## Scope

The following are in scope:

- The extension source code (`extension.js`)
- The color theme JSON
- The webview assets and dashboard
- The build and packaging process

The following are out of scope:

- VS Code itself
- Third-party extensions (GitLens, Git Graph, Error Lens)
- The underlying operating system
